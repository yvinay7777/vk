# AI Resume Navigator MVP

A modern resume navigator built with Next.js 14, Tailwind CSS, Firebase, OpenAI, Remotive, and RemoteOK.

## Features

- Email authentication with Firebase
- Resume upload (PDF / DOCX)
- Resume parsing, skills and education extraction
- Job fetching from Remotive and RemoteOK
- Job matching and AI-powered improvement suggestions
- Resume dashboard and saved jobs

## Setup

1. Copy `.env.example` to `.env.local`
2. Fill Firebase and OpenAI values
3. Install dependencies:

```bash
npm install
```

4. Run the dev server:

```bash
npm run dev
```

## Deploy

Deploy with Vercel or Firebase Hosting. Ensure environment variables are set.
